# 知的UIデザインコンセプト

## 現状分析
現在のデザインは以下の特徴があります：
- カラフルで親しみやすい配色（ブルー、ピンク、オレンジなど）
- 大きめのボタンとポップなスタイル
- イラストを使った視覚的アプローチ

## 新デザインコンセプト「知的・信頼性・プロフェッショナル」

### 1. カラースキーム
```css
/* 知的で信頼感のある配色 */
--primary-navy: #1a2332;        /* 濃紺 - メインカラー */
--primary-blue: #2c3e50;        /* ダークブルーグレー */
--accent-gold: #b8860b;         /* ゴールド - アクセント */
--accent-blue: #3498db;         /* 明るいブルー - CTA */
--text-primary: #2c3e50;        /* 本文 */
--text-secondary: #7f8c8d;      /* 補助テキスト */
--bg-primary: #ffffff;          /* 背景白 */
--bg-secondary: #f8f9fa;        /* 薄いグレー背景 */
--bg-tertiary: #e9ecef;         /* グレー背景 */
--border-light: #dee2e6;        /* ボーダー */
```

### 2. タイポグラフィ
```css
/* セリフ体とサンセリフ体の組み合わせ */
--font-heading: 'Noto Serif JP', 'Georgia', serif;     /* 見出し用 */
--font-body: 'Noto Sans JP', 'Helvetica Neue', sans-serif;  /* 本文用 */
--font-mono: 'IBM Plex Mono', monospace;               /* データ表示用 */

/* フォントサイズ階層 */
--fs-h1: clamp(2rem, 4vw, 3rem);
--fs-h2: clamp(1.5rem, 3vw, 2.25rem);
--fs-h3: clamp(1.25rem, 2.5vw, 1.75rem);
--fs-body: clamp(1rem, 1.5vw, 1.125rem);
--fs-small: 0.875rem;
```

### 3. レイアウト原則
- **グリッドベース**: 8pxグリッドシステム採用
- **黄金比**: 重要な要素に1:1.618の比率を適用
- **余白**: 十分な余白で読みやすさと高級感を演出
- **左右対称**: バランスの取れた対称的なレイアウト

### 4. UI要素のスタイル

#### ボタン
- フラットデザインから立体的なデザインへ
- ホバー時に微細なシャドウとスケール変化
- プライマリボタンは深い青、セカンダリは枠線のみ

#### カード
- 薄いシャドウと白背景
- コンテンツに集中できるミニマルな枠線
- ホバー時に浮き上がるエフェクト

#### フォーム
- 大きめの入力フィールド
- フォーカス時に青いアウトライン
- ラベルは上部に配置

### 5. アニメーション
- **控えめ**: 0.2-0.3秒の短いトランジション
- **意味のある動き**: ユーザーの操作を補助
- **イージング**: cubic-bezier(0.4, 0, 0.2, 1)を基本に

### 6. アイコン・画像
- **ラインアイコン**: シンプルで洗練されたアイコン
- **モノトーン写真**: 必要に応じて白黒またはセピア調
- **データビジュアライゼーション**: チャートやグラフで情報を視覚化
