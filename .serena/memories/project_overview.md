# Talent-Tuner プロジェクト概要

## プロジェクト目的
職業適性診断アプリケーション「ため職」（talent-tuner）は、ユーザーの質問への回答に基づいて最適な職業を推奨するVue.jsアプリケーションです。

## 主要機能
- **職業適性診断**: 4つのカテゴリー（スキル、興味、優先事項、バランス）で16の質問による診断
- **AI進路相談**: Claude AI APIを使用したチャットbot機能
- **職業詳細情報**: 28職業の詳細データベース
- **多言語対応**: 日本語、英語、中国語（Vue I18n使用）
- **Stripe決済**: 詳細PDFレポートの販売機能
- **25ページ**: AdSense審査対応の豊富なコンテンツ

## 技術スタック
- **フロントエンド**: Vue 3 + TypeScript (Composition API)
- **ビルドツール**: Vite 6.2.4
- **ルーティング**: Vue Router 4.5.0
- **国際化**: Vue I18n 10.0.4
- **状態管理**: コンポーネントローカル状態（Pinia 3.0.1利用可能）
- **スタイリング**: CSS Variables + レスポンシブデザイン
- **デプロイ**: お名前.com共用サーバー（自動化スクリプト）

## 対象環境
- **開発環境**: Node.js 20+、npm
- **本番環境**: お名前.com共用サーバー（PHP対応）
- **CI/CD**: GitHub Actions
- **外部API**: Claude AI、Stripe

## データ管理
- **静的JSONファイル**: `/public/data/` に診断・職業データ
- **動的読み込み**: fetch APIでクライアントサイドデータ取得
- **言語別データ**: 多言語コンテンツの分離管理

## アーキテクチャ特徴
- **SPA**: Vue Routerでクライアントサイドルーティング
- **コンポーネント志向**: 再利用可能なVueコンポーネント
- **型安全**: TypeScript完全対応
- **SEO対応**: 動的メタタグ、構造化データ
- **パフォーマンス**: コード分割、遅延読み込み
